# Install land-cli

This document describes how to install the `land-cli` command line tool.

## Pre-requisites

:::tip

- `land-cli` requires Rust 1.68 or later
- `rustup target add wasm32-wasi` is required to build the webassembly binary

:::

## Install

`land-cli` runs on Linux (x86_64 and aarch64), macOS (x86_64 and aarch64). Windows is not tested yet.

<Tabs>
  <TabItem value="linux" label="Linux" default>
<p>The easiest is to use the installer script, hosted on this site:</p>

If you have `cargo` installed.

```bash
cargo install land-cli --git https://github.com/fuxiaohei/runtime.land
```

After installing, you can check if it was successful by running:

```bash
land-cli --version
```

  </TabItem>
  <TabItem value="macos" label="MacOS">

If you have `cargo` installed.

```bash
cargo install land-cli --git https://github.com/fuxiaohei/runtime.land
```

After installing, you can check if it was successful by running:

```bash
land-cli --version
```

  </TabItem>
  <TabItem value="windows" label="Windows">

:::caution

Windows binary is not available yet.

:::

  </TabItem>
</Tabs>


## Building from source

Then clone the source code and build it:

```bash
git clone https://github.com/fuxiaohei/runtime.land
cargo install --locked --path .
land-cli --help
```
